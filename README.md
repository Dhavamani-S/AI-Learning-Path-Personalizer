# AI-Based Learning Path Personalizer

## Overview
This project provides users with a personalized learning roadmap tailored to their profile, including education level, target field, experience, duration, and desired mastery level. The output roadmap includes topics, subtopics, resource links, and estimated learning hours in a clear table format.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI Module:** Python (Content-Based Recommendation + Rule-Based Filtering)

## Features
- User input forms to collect learning preferences
- AI-driven personalized roadmap generation
- Admin panel for managing learning resources
- Display roadmap in a day-wise table format

## Getting Started
1. Clone the repository
2. Set up backend and frontend (instructions coming in future steps)
3. Run server and frontend locally
